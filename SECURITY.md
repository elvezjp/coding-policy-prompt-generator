# Security Policy

## Supported Versions

This project is under active development. Security fixes will be applied to the latest version on the default branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report vulnerabilities privately:

- Email: info@elvez.co.jp
- Subject: `[SECURITY] coding-policy-prompt-generator`

Please include:

- A clear description of the issue
- Steps to reproduce
- Impact assessment (what could go wrong)
- Suggested mitigation, if known

We will acknowledge receipt and aim to provide an initial response within 7 business days.
